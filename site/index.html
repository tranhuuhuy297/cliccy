Keyboard-first

Type to filter, arrows to move, Alt+1–9 to quick-pick.

Text & images

Captures copied text and PNG images, with thumbnails inline.

Pin snippets

Pin with Ctrl+P. Pinned items never expire.

Zero jitter

Event-driven via X11 XFIXES — reads only on real changes.

No dock icon

A clean popup that auto-hides on focus loss — no taskbar clutter.

X11 & Wayland

One small binary, SQLite-backed, dark Catppuccin theme.
